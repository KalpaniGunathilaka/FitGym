# 💪 FitGym

✨ FitGym is a modern fitness website built using React and Tailwind CSS.

---

## 🚀 Features

- 🌙 Dark / Light mode  
- 📱 Responsive design  
- 🎨 Modern UI  
- 🔍 Services search filter  
- 📩 Contact form  

---

## 🛠️ Tech Stack

- ⚛️ React  
- 🎨 Tailwind CSS  
- 🎬 Framer Motion  
- 🔗 React Icons
